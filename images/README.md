# Behavioral Insights Images

This folder contains images displayed in the Behavioral Insights section of the Streamlit app.

## How to Save the Images

Each image has been provided. Follow these steps to save them to this folder:

### Option 1: Manual Save (Windows)
1. Right-click on each image below
2. Select "Save image as"
3. Navigate to `c:\Users\Win11\Engagement_Score_Predictor\images\`
4. Save with the exact filename listed below

### Required Images & Filenames

1. **distribution_engagement_tiers.png** 
   - Bar chart showing count of users in each engagement tier

2. **messages_sent_engagement.png** 
   - Box plot: Messages Sent Count vs. Engagement Tier

3. **swipe_right_ratio_engagement.png** 
   - Box plot: Swipe Right Ratio vs. Engagement Tier

4. **app_usage_time_engagement.png** 
   - Box plot: App Usage Time (Minutes) vs. Engagement Tier

5. **emoji_usage_engagement.png** 
   - Box plot: Emoji Usage Rate vs. Engagement Tier

6. **relationship_intent_engagement.png** 
   - Grouped bar chart: Relationship Intent Impact on Engagement Tier

## Verification

Once you've saved all images, restart the Streamlit app and the Behavioral Insights page will display them correctly.
